## The 5 Problem Statements — Scored for a 2-Person Team

### 🥇 #1 Recommendation: **Unbiased AI Decision**

**Why this wins:**
- Google *deeply* cares about Responsible AI — this aligns with their core mission
- Extremely high technical merit ceiling (bias detection pipelines, fairness metrics, explainability)
- Highly original — most teams won't touch this
- Perfect for Gemini API usage (LLM bias auditing, explanation generation)
- Strong future potential story (societal impact at scale)

**Project Idea: "FairLens" — AI Bias Detection & Mitigation Platform**
- Upload any AI dataset or model output → AI analyzes for bias across gender, age, caste, geography
- Generates a **bias report** with explainability (why the bias exists)
- Suggests mitigation strategies using Gemini
- Dashboard showing fairness metrics (demographic parity, equalized odds)
- Real-world use case: loan approvals, hiring systems, medical diagnosis

**Score Potential:**
- Technical Merit: ★★★★★ (Gemini API + fairness algorithms + visualization)
- Alignment: ★★★★★ (massive real-world problem in India specifically)
- Innovation: ★★★★★ (very few student teams do this well)
- UX: ★★★★☆ (dashboard + report UI — manageable for 2 people)
